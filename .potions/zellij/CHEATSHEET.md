POTIONS · Zellij Cheatsheet
===========================

Press q to close.

PREFIX  ─  Ctrl+a, then…
------------------------
  Split        |   right     -   down
  Navigate     h j k l       (vim-style)
  Resize       Ctrl+h/j/k/l  (one-shot) · r (resize mode)
  Tabs         c new   x close pane   X close tab
               n next  p prev   1…9 jump
  Pane         z fullscreen   f float   F toggle floating
               { } shuffle    < > swap
               B break into tab
  Layout       = / E / T / M / V  (cycle)
  Session      S  ·  Esc cancel
  Send ^A      Ctrl+a (twice)

NO PREFIX
---------
  Panes        Alt+h/j/k/l  or  Alt+←/↓/↑/→
  Split        Alt+d        (right)
  Tabs         Alt+t new   Alt+w close
               Alt+n next  Alt+p prev
  Fullscreen   Alt+Enter

CHEATSHEET
----------
  Alt+/        open this cheatsheet (normal mode)
  Ctrl+a ?     open this cheatsheet (tmux mode)
