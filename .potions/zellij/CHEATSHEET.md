POTIONS · Zellij Cheatsheet
===========================

Press q to close.

Works on EVERY platform with no setup: the Ctrl+a prefix below, F1, or
Ctrl+a then ? — all open this cheatsheet. On macOS, "Alt" means Option (⌥).

PREFIX  ─  Ctrl+a, then…   (universal — always works)
------------------------
  Split        |   right     -   down
  Navigate     h j k l       (vim-style)
  Resize       Ctrl+h/j/k/l  (one-shot) · r (resize mode)
  Tabs         c new   x close pane   X close tab
               n next  p prev   1…9 jump
  Pane         z fullscreen   f float   F toggle floating
               { } shuffle    < > swap
               B break into tab
  Layout       = / E / T / M / V  (cycle)
  Session      S  ·  Esc cancel
  Help         ?  open this cheatsheet
  Send ^A      Ctrl+a (twice)

NO PREFIX  (convenience layer — see macOS note)
-----------------------------------------------
  Panes        Alt+h/j/k/l  or  Alt+←/↓/↑/→
  Split        Alt+d        (right)
  Tabs         Alt+t new   Alt+w close
               Alt+n next  Alt+p prev
  Fullscreen   Alt+Enter
  macOS        Alt = Option (⌥). These fire only with "Use Option as Meta"
               (or Cmd→Alt forwarding). Otherwise use the Ctrl+a prefix above —
               it does everything here. Setup: ~/.potions/terminal-setup/

CHEATSHEET
----------
  F1           open this cheatsheet (any platform, no setup)
  Ctrl+a ?     open this cheatsheet (universal — via prefix)
  Alt+/        open this cheatsheet (needs Option-as-Meta on macOS)

Full reference: ~/.potions/KEYMAPS.md
